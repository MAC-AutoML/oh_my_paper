# Artifact contract

Core workspace files:

- `.paper-ai/PAPER_AI_STATE.md`: current phase, blockers, next action.
- `.paper-ai/TRACE.jsonl`: normalized event stream, implemented in Milestone 2.
- `.paper-ai/MATERIALS_USED.md`: public-safe material references.
- `paper/CLAIMS.md`: claim IDs and support status.
- `paper/EVIDENCE_MAP.md`: evidence, missing evidence, caveats.

If an artifact is missing, create a minimal placeholder rather than relying on chat memory.
