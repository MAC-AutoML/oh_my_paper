# Deep Research guide

This guide anchors the research-side oh my paper workflow in Codex-native skills and agents.

1. Start with a stable research question, not a title.
2. Record the intended venue, field, and evidence standard.
3. Separate exploratory ideas from supported claims.
4. Build a search plan before drafting prose.
5. Prefer primary papers, datasets, benchmark reports, and official docs.
6. Mark blog posts and secondary summaries as context only.
7. Use Semantic Scholar verification for every citation candidate when enabled.
8. In no-key mode, expect slower calls and rely on cache reuse.
9. In disabled mode, label citation verification as skipped rather than clear.
10. Never treat a model-suggested reference as real until verified.
11. For each candidate source, track title, authors, year, DOI or paper id, and search status.
12. Classify citations as verified, ambiguous, not found, rate limited, error, or skipped.
13. Ambiguous matches require author/year review before writing uses them as support.
14. Not-found sources may be mentioned only as user-provided unverified notes.
15. A literature corpus should include source quality, relevance, claim links, and limitations.
16. A PRISMA-style review should include search strings, inclusion rules, exclusion rules, and counts.
17. The Socratic mentor lane should ask why the problem matters and what would falsify the idea.
18. The Devil's Advocate lane should challenge novelty, feasibility, and evidence sufficiency.
19. The source verifier lane should avoid judging prose quality; it certifies source existence and match quality.
20. The research lead consolidates only verified or explicitly caveated evidence.
21. Handoff to writing only when a research brief and citation status report exist.
22. Handoff notes must name missing evidence, not hide it in polished prose.
23. If the user supplies private materials, keep raw files outside packaged skill content.
24. If a live API fails, produce an honest not-run or rate-limited status.
25. The output should make uncertainty visible enough for the writing skill to avoid overclaiming.
